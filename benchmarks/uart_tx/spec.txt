Design: UART Transmitter (TX only)
Module name: uart_tx
Inputs: clk, rst_n, tx_start (1-bit), tx_data (8-bit), baud_div (16-bit)
Outputs: tx (1-bit serial output), tx_busy (1-bit)
Behavior:
  When tx_start=1 and tx_busy=0, begin transmission
  Transmit: 1 start bit (0), 8 data bits LSB first, 1 stop bit (1)
  tx_busy=1 during transmission, 0 when idle
  baud_div sets clock divider for baud rate generation
  After stop bit, return to idle state
  rst_n is active-low synchronous reset
